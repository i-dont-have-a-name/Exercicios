# 2º EXERCICIO 

## Contador de passos

* **Faça um programa em que**
  1. - *Peça 3 valores*
  2. - *Primeiro valor, a posição atual*
  3. - *Segundo valor, o destino*
  4. - *Terceiro valor, quanto o passo adiciona até chegar ao destino*
  
  
##  __EXEMPLO__

* **Valores de entrada**
> 5, 10, 2

* **Saida**

> `A quantidade de passos dados será 3`
